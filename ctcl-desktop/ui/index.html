自訂時鐘

建立一個以固定倍率走的時鐘——遊戲世界、模擬加速、或任何跟現實時間不同步的時間軸。目前只支援固定倍率（進階的分段／暫停／查表時鐘，可以透過 CLI 或 API 建立）。

時間群組

把好幾個時區或自訂時鐘綁在一起，一次展開同一個瞬間在每個成員裡是什麼時間——「一瞬間，多世界」。

本機 API …

讓其他你信任的程式或 agent 呼叫這台電腦——預設關閉，只能從這台電腦自己存取，其他電腦連不到。

裝置時鐘觀測 …

在背景定期比對這台電腦的時鐘——偵測睡眠／喚醒、時鐘被手動調整、或跟正常速度不一致的時鐘漂移。只會記錄異常，不會把一切正常也寫進紀錄。

觸發器 …

時間到了、或某個自訂時鐘走到某個點的時候，自動記一筆通知或打開一個 URI（交給作業系統決定要開哪個 App）。每個觸發器只會觸發一次。

開啟觸發器引擎

WakeEvents（Agent 喚醒紀錄）

「喚醒 Agent」觸發器觸發時，CTCL 只會在這裡留一筆紀錄，不會替你呼叫任何工具或做任何決定 - 實際要不要行動、做什麼，是外部 Agent 自己的事。這裡可以手動標記「已確認收到」。
